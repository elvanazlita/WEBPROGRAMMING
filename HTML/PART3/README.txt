MENGENAL STYLE, FORMATTING, DAN COMMENT DI HTML

STYLE PADA HTML;
    Style pada html digunakan untuk mengatur tampilan elemen-elemen di halaman web. Dengan menggunakan style, kita bisa menentukan warna, ukuran font, tata letak, dan berbagai properti visual lainnya dari sebuah elemen. Dan umumnya, untuk mendefinisikan style dalam HTML yaitu kita bisa menggunakan CSS (Cascading Style Sheets).

CARA MENAMBAHKAN STYLE ATAU CSS KE ELEMEN HTML;
    1.  Inline Style:
        Style langsung dituliskan di dalam tag HTML elemen yang bersangkutan dengan menggunakan atribut style.
    2.  Internal Style Sheet:
        Style dituliskan di dalam tag <style> yang ditempatkan di dalam bagian <head> dari HTML.
    3.  External Style Sheet:
        Style dituliskan dalam file terpisan dengan ekstensi 
        .css dan kemudian di hubungkan ke file HTML menggunakan tag <link>.

FORMATTING PADA HTML;
    Formatting pada HTML digunakan untuk mengatur tampilan teks dan elemen-elemen lain di halaman web agar terlihat menarik, terbaca, dan sesuai dengan tujuan kita. Formatting dilakukan dengan menggunakan tag-tag HTML khusus yang membuat kita bisa membuat teks menjadi tebal, miring, bergaris bawah, dll.

CONTOH TAG HTML UNTUK FORMATTING TEKS;
    1.  <b> : membuat teks menjadi tebal.
    2.  <i> : membuat teks menjadi miring.
    3.  <u> : membuat teks menjadi bergaris bawah.

COMMENT ATAU KOMENTAR PADA HTML;
    Komentar pada HTML adalah bagian dari kode yang tidak akan ditampilkan di halaman web saat diakses. Dan inilah beberapa kegunaan komentar pada HTML:
        1.  Memberikan penjelasan
        2.  Menonaktifkan sementara bagian kode
        3.  Meningkatkan keterbacaan kode
