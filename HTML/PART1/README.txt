MENGENAL DASAR-DASAR HTML:

DEFINISI HTML;
        - HTML sendiri merupakan singkatan dari HyperText Markup Language yang biasanya digunakan untuk membuat struktur halaman web.

STRUKTUR HTML;
        - HTML bisa di bayangkan seperti sebuah kerangka rumah. Dimana kerangka tersebut yang nantinya akan menunjukkan di mana letak pintu, jendela, dan ruangan-ruangan yang lain. 

TAG HTML;
    - HTML terdiri dari berbagai elemen yang disebut tag. Tag-tag ini diawali dan diakhiri dengan tanda kurung siku (<>). Setiap tag memiliki fungsi spesifik untuk menentukan tampilan dan struktur konten pada halam web.
    - Mengenal beberapa tag HTML dan fungsinya;
        1. Tag <h1> hingga <h6> : dalam HTML, tag ini digunakan untuk mendefinisikan judul atau heading pada suatu halaman web. Setiap tag memiliki tingkatan yang berbeda, mulai dari <h1> sebagai judul utama atau yang paling besar, hingga <h6> sebagai subjudul terkecil.
        2. Tag <p> : Tag ini digunakan untuk membuat paragraph.
        3. Tag <img> : Tag ini digunakan untuk menyisipkan gambar ke dalam halaman.
        4. Tag <a> : Tag ini digunakan untuk membuat link atau tautan ke halaman lain.
