PENGGUNAAN COLOR, UNORDERED LIST, DAN ORDERED LIST PADA HTML

PENGGUNAAN COLOR PADA HTML;
    Color atau warna adalah elemen penting dalam desain web. Color tidak hanya membuat tampilan website menjadi lebih menarik, tetapi juga dapat menyampaikan pesan tertentu dan membangkitkan emosi pada pengguna.

CARA MENAMBAHKAN ATAU MENULISKAN COLOR PADA HTML;
    1.  Langsung menggunakan nama color
    2.  Menuliskan kode hexadesimal
    3.  Menuliskan kode RGB
    4.  Menuliskan kode HSL
    5.  Menuliskan kode HSV

    https://colorpicker.me

MENGENAL APA ITU UNORDERED LIST PADA HTML;
    Unordered list pada HTML digunakan untuk menampilkan daftar item yang tidak memiliki urutan tertentu. Biasanya setiap item dalam daftar akan diawali dengan tanda bulatan kecil.

CARA MEMBUAT UNORDERED LIST:
    1.  menggunakan tag <ul> untuk memulai dan mengakhiri daftar
    2.  Menggunakan tag <li> untuk setiap item atau list dalam daftar

MENGENAL APA ITU ORDERED LIST PADA HTML;
    Ordered list pada HTML digunakan untuk menampilkan daftar item yang memiliki urutan tertentu. Setiap item dalam daftar bisa diawali dengan nomor atau huruf.

CARA MEMBUAT ORDERED LIST:
    1.  menggunakan tag <ol> untuk memulai dan mengakhiri daftar
    2.  Menggunakan tag <li> untuk setiap item atau list dalam daftar

CONTOH ATRIBUT & TYPE PADA TAG ORDERED LIST <ol>:
    -   1 : item akan diawali dengan angka 1 sampai seterusnya
    -   A : item akan diawali dengan huruf A (huruf besar latin) sampai seterusnya
    -   a : item akan diawali dengan huruf a (huruf kecil latin) sampai seterusnya
