MENGENAL ENTITIES, BREAKLINE, DAN HORIZONTAL RULE;

ENTITIES HTML;
    Entities pada HTML adalah representasi khusus dari karakter yang memiliki makna khusus dalam kode HTML atau karakter yang tidak memiliki representasi langsung pada keyboard.
    
    Contoh website penyedia karakter entities;
    https://www.w3schools.com/html/html_entities.asp

BREAKLINE;
    Breakline atau lebih tepatnya tag <br> adalah elemen HTML yang digunakan dalam sebuah paragraf. Tag ini membantu untuk merapikan teks agar mudah di baca terutama untuk penulisan teks yang membutuhkan pemisahan baris secara spesifik.

HORIZONTAL RULE;
    Horizontal Rule atau lebih tepatnya tag <hr> adalah elemen HTML yang digunakan untuk membuat garis horizontal pada halaman web. Garis ini sering digunakan untuk memisahkan bagian-bagian konten yang berbeda.
